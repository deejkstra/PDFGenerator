# PDFGenerator
Generate a PDF from the command line.

```
./pdf_generator.py --description "Test test" --docs bs:1 fm:2 ps:3
```

## Requirements

* ImageMagick
* PDFtk

### Mac

```
brew update && brew install imagemagick
brew install pdftk
```

### Ubuntu

```
sudo apt install imagemagick
sudo apt install pdftk
```
